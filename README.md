# First Eshop

A demo e-shop built with HTML, CSS, and Vanilla JavaScript.  
The project includes product listing, filtering, sorting, search, pagination, cart functionality, and cost calculation.

## Features

- Product filtering by category
- Product sorting by:
  - Recommended
  - Price ascending
  - Price descending
- Product pagination
- Select how many products appear per page
- Search products with a search bar
- Add products to cart
- Select product quantity
- Cart page with selected products
- Cart total cost calculation
- Cart data saved with LocalStorage

## Technologies

- HTML5
- CSS3
- JavaScript
- LocalStorage

## Project Status

This is a demo project created for practice and portfolio purposes.
